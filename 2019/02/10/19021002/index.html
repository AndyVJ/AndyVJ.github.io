三条路

发表于 2019-02-10 | 分类于 其他 | 评论数： | 阅读次数：

现在要学会做减法，要规划一下自己，结合自己的资源，最有可能成功的一条路径。在没有特别好机会的情况下三年内按这个方向走。

产品方向

建模造型 + 模流分析 + 电子。

建模造型

UG 或 Creo 的(曲面)建模造型能力，主要是图片造型能力。

模流分析

主要是常用产品材料属性，产品分型。

电子

产品肯定需要电子，非专业以了解为主。

可能的结果

  1. 成为软件专家方向。做教程，卖教程；做自己的网站论坛，做培训等；
  2. 产品结构专家。造型能力 + 材料属性 + 模流分析；
  3. 研发销售一体化。前期代理销售 -> 仿造 -> 规模做大 -> 成立公司。

机械方向

其实，越来越觉得这不是一个很好的反向。本着“鸡蛋不能放在同一个篮子里的原则”，这个方向还是很有必要的。

机械原理

各种机构的熟悉，运动仿真等。

电气 PLC

以销售二手 PLC 为原点，窥探行业机会。电气控制专业化去服务，挖掘一些客户；如果这样可行，会有一些机会。

上位机SCADA

非专业，做兴趣定位。

可能的结果

  1. 机械结构 + 电气控制 + 上位机 全栈专家；
  2. 有做公司的机会。

工作职业

做好现有的工作，成为专家。工作中其实大的方向上都含有上面两部分。

主要机会在于：

  1. 机器本身；
  2. 服装业。

总结

围绕上面展开，需要具体做的事情有很多，做下减法主要是下面几个：

  1. 造型；
  2. 机械结构运动仿真 + PLC；
  3. 店开起来，维护起来，推广起来。
  4. 电子电气，编程知识就放在兴趣方面吧。
感谢支持！
