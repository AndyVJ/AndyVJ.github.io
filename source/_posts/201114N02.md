## 控制台输出

```Csharp
Console.WriteLine("{0:F2}", 10.2658749);
```

输出：

```
10.26  // 可见其是直接将后面的小数位舍弃了。
```

《C# 图解教程》 P19 页对格式化输出有详细的介绍。

## `Math.Round`函数进行四舍五入

```Csharp
double a = 1.223467;
a = Math.Round(a, 4);
Console.WriteLine(a);
```
输出：
```
1.2235
```
